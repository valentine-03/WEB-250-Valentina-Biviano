Selenium Testing

In this session, I explored automated user interface testing using Selenium IDE to record and verify functional workflows within my web applications. By testing my CookBookie (recipe organizer) project, I learned how to create test scripts that automatically simulate user actions like filtering recipe categories, navigating the dashboard, and validating form submissions for adding new recipes. This experience highlighted the importance of regression testing in server-side scripting to ensure that new code updates do not break existing features. As I move forward with my educational goals at Harper College, I intend to apply these testing methodologies to build more robust, error-free applications and improve the overall reliability of my web development projects for future career opportunities.
