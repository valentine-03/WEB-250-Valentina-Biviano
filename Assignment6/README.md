Module 6 WEB 250

Activity #4 Description: Create an application that allows the user to select 
and upload a comma-separated-values file containing dates, tsunamis, continents, 
and countries. Build a two-dimensional array of tsunami data. Display the tsunami 
information in alphabetical order by continent and country. Use CSS formatting to 
color-code the data based on continent.

In this assignment, I learned how server-side scripting can process user-uploaded 
files and manipulate their data dynamically using arrays. I built a Node.js Express 
application that reads a CSV file, stores the data in a two-dimensional array, sorts it, 
and displays it with conditional formatting. This helped me understand a bit more how 
backend logic interacts with frontend display!
